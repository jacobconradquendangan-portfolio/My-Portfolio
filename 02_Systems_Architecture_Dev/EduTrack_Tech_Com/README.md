## Project Overview
EduTrack Tech Com is a web-based enrollment prototype engineered to modernize the CIIT enrollment experience. This project serves as the final technical output for the Technical Communication course, translating UX research and survey data into a functional, student-centric digital solution.

### Key Problem Solved:
* **System Stability:** Built as a lightweight Vanilla JS application to demonstrate a crash-resistant architecture.
* **Transparency:** Surfaces instructor ratings and detailed performance metrics before enrollment.
* **Financial Clarity:** Provides a real-time breakdown of tuition fees and institutional charges.

## Technical Implementation
This prototype is built using a "No-Framework" approach to demonstrate mastery of core web technologies:
* **Frontend:** HTML5, CSS3 (Custom Variables & Grid/Flexbox).
* **Logic:** Vanilla JavaScript for state management (Subject Cart) and DOM manipulation.
* **Persistence:** Utilizes `localStorage` for session handling and user personalization.

## How to Launch
1. Open the root folder in your code editor.
2. Launch `login.html` in a web browser.
3. Use a CIIT institutional email to bypass the authentication gateway.
